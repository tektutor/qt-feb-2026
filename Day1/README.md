# Day 1

## Info - Qt Framework
<pre>
- Qt Framework is developed in C++
- is a cross-platform GUI application framework
- Qt comes in 2 flavours
  - Community Edition
  - Paid version for Commercial use - comes with Support
- Qt Frameworks supports developing GUI Application using C++/Python
- Qt Frameworks two major frameworks
  - Qt Widget Framework
    - this is very useful to devel GUI application natively in C++
    - the application look and feel will look normal just like how windows applications are looking
  - QML Framework
    - this supports developing impressive application in terms of Look & Feel
    - supports Fluid custom widgets
    - the language used is QML - Qt Markup Language ( an extension of JavaScript )
</pre>
